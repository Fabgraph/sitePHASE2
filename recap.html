Propriétés CSS utilisées dans cette page

Décrivez leur usage et leurs valeurs possibles

  1. background: couleur du fond de l'élément
    1. background applat ou linear gradient pour un dégradé
    2. elle peut être appelée par son nom ex. white, pink; ou des valeurs de couleur web ex. #fff (blanc) ou #333 (gris); ou en rgba ex. rgba(0, 0, 0, 0.1)
  2. color: couleur de la police
  3. float: effet flottant
    1. elle peut être float left pour flotter à gauche
    2. ou float right pour flotter à droite
    3. il faut une div class="clear" pour remettre le flux dans la HTML5 et dans le CSS clear: both
  4. font-family: police
  5. font-size: taille de la police
    1. la taille de la police est en em ex. font-size: 1.2em;
  6. height: hauteur d'un élément
    1. elle peut être en px
    2. ou en pourcentage par rapport à l'élément qui la contient
  7. left: position à partir de la gauche de la fenêtre
  8. line-height: hauteur de l'interligne
  9. margin: marges extérieures
    1. elles ont pour valeurs des px, em ou %
  10. padding: marges intérieures
    1. sert à déplacer ce qu'il y a dans le bloc; ex. à écarter un texte du bord du bloc
    2. elles ont pour valeurs des px, em ou %
  11. position: position de l'élément dans la page (on n'a utilisé que la valeur (fixed) pour l'instant)
  12. text-align: alignement du texte
    1. il peut être right à droite
    2. ou left à gauche
    3. ou center au centre
    4. ou justifier avec justify
  13. text-decoration: changement des valeurs héritées (ici le souligné qui est standard pour les liens sous tous les navigateurs est supprimé)
    1. none permet de supprimer le soulignement des liens
  14. top:
      position à partir du haut de la fenêtre
    1. 0 auto; l'élément est au centre en haut
    2. auto; l'élément est au centre de la page
  15. transition: temps de transition lors d'une animation (ici sur les liens de la barre de menus)
  16. width:
    • largeur de l'élément par rapport à la taille de la fenêtre si on indique une valeur relative (%, em, ...)

Propriétés CSS à rechercher

Recherchez les propriétés ci-dessous, faites des tests avec sur cette page depuis la console de débuggage pour tester les valeurs qu'elles peuvent prendre

  1. font-weight:
    • gras.
  2. letter-spacing:
    • définit l'interlettre utilisée pour les caractères qui composent le texte.
  3. word-spacing:
    • définit l'espace entre les mots.
  4. text-shadow:
    • ombre de texte.
  5. background-image:
    • permet de définir une ou plusieurs images comme arrière(s)-plan(s) pour un élément.
  6. background-size:
    • définit la taille des images d'arrière-plan pour l'élément. La taille de l'image peut être contrainte, complètement ou partiellement afin de conserver ses proportions.
  7. opacity:
    • définit la transparence d'un élément. Autrement dit, elle permet de définir le degré de visibilité de l'arrière-plan sur lequel est placé l'élément.
  8. border-style: permet de donner une bordure à un élément.
  9. box-shadow:
    • permet de donner une ombre à la boîte d'un élément.
  10. list-style-image:
    • puce personnalisée.
  11. border:
    • c'est une propriété raccourcie qui permet de définir les propriétés liées à la bordure. border peut être utilisée pour définir les valeurs de border-width, border-style, border-color.
  12. border-radius: permet de définir des coins arrondis pour la bordure d'un élément.
  13. border-width:permet de donner une bordure la boîte d'un élément.
  14. font-variant: petites capitales.
  15. background-color: permet de définir la couleur utilisée pour l'arrière-plan d'un élément (celle-ci peut être une couleur transparente).
  16. !important;: permet de spécifier que la propriété est la plus importante dans votre feuille CSS en lui donnant un poids plus important. La règle de feuille de style !important CSS est généralement utilisée dans le cas où une propriété de feuille de style CSS est déclarée plusieurs fois avec des ciblages différents.
